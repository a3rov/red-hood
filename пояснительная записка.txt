Основной файл запуска игры - game.py

Из него запускается файл world.py или menu.py (в зависимости от статуса игры).

В папке core расположены главные файлы игры:
world (папка) - для работы с миром
ui (папка) - содержатся все объекты (Label, Button, InviseButton и так далее) для дизайна игры.
objects - все объекты, которые нужны для работы игры (персонажи, небо, звуки, блоки)
sql.py - файл для работы с базой данных.

В основной папке расположено:
start.py - файл, в котором редактируется один из уровней (он не входит в проект, так как создан для себя)

в папке core/world расположены файлы для основной работы мира:
physic.py - работа с физикой
camera.py - работа с камерой
world.py - работа с миром

В каждом файле, где есть методы прописано что они делают

В Папке resource хранятся все картинки, звуки, шрифты, база данных, уровни, настройки
